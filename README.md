# PL교회 교리교육 조 배치 검색기 🚀

## 📋 프로젝트 소개

교회 교리교육 성도들이 자신의 조 배치 정보를 이름과 전화번호 뒷자리를 통해 쉽고 빠르게 찾을 수 있도록 개발된 웹 시스템입니다. 구글 스프레드시트와 실시간 연동되며, 구글 드라이브의 이미지 호스팅 제한을 우회하는 기술적 처리가 적용되었습니다.

## ✨ 주요 기능

### 1. 일반 사용자 모드

* **실시간 데이터 조회**: 구글 스프레드시트(CSV)를 DB로 사용하여 업데이트가 즉각 반영됩니다.
* **간편 검색**: 이름 + 전화번호 뒷 4자리 조합으로 조와 교육 장소를 확인합니다.
* **배치도 시각화**: 장소에 맞는 지도를 자동 표시하며, 클릭 시 확대(모달) 기능을 제공합니다.
* **테마 전환**: 사용자 편의를 위한 다크/라이트 모드를 지원합니다.

### 2. 튜터 / 서브튜터 전용 기능 ⭐

* **조원 명단 자동 노출**: 권한(`role`)이 '튜터' 혹은 '서브튜터'인 사용자가 로그인하면 하단에 **전체 조원 명단**이 표시됩니다.
* **식별 편의성**: 조원 이름 뒤에 번호가 붙어 표시됩니다 (예: `이민재6550`).
* **권한 제어**: 일반 조원이 검색할 때는 해당 명단이 노출되지 않습니다.

### 3. 관리자 모드

* **전용 관리 페이지**: 별도의 관리자 로그인을 통해 전체 명단 및 통계를 확인합니다.
* **접속 정보**: ID: `plc` / PW: `plc1234`

# PL교회 교리교육 조 배치 검색기

## 📋 프로젝트 소개
교회 교리교육 성도들이 자신의 조 배치 정보를 쉽게 찾을 수 있도록 만든 웹 검색 시스템입니다.

## ✨ 주요 기능

### 일반 사용자 모드
- 이름 + 전화번호 뒷 4자리로 조 검색
- 조, 위치 정보 표시
- 조별 배치도 이미지 표시 및 확대 보기
- 다크/라이트 테마 전환
- 60세 이상 자동 큰 글씨 모드
- 모바일 최적화된 반응형 디자인

### 관리자 모드 (신규)
- **검색 모드**: 이름만으로 검색 가능 (동명이인 처리)
- **조별 보기**: 모든 조를 카드 형식으로 표시 및 조원 목록 확인
- **개인별 보기**: 전체 조원 명단을 카드 형식으로 표시
- 실시간 검색/필터링 기능
- 관리자 계정 로그인 (plc / plc0110)

## 🚀 사용 방법

### 일반 사용자
1. 웹사이트 접속
2. 이름과 전화번호 뒷 4자리 입력
3. "조 확인하기" 버튼 클릭
4. 조, 위치, 배치도 확인

### 관리자
1. 우측 상단 관리자 아이콘 클릭
2. 아이디: `plc`, 비밀번호: `plc0110` 입력
3. 3가지 모드 중 선택하여 사용:
   - **검색 모드**: 이름으로 빠른 검색
   - **조별 보기**: 조 단위 관리
   - **개인별 보기**: 전체 명단 확인

### 로컬 서버 실행 (개발용)
```bash
python -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000` 접속

## 📝 데이터 업데이트 방법

조배치 변경, 참여자 추가, 하차자 처리 등의 변화가 있을 때 다음과 같이 업데이트합니다.

### 1. CSV 파일 준비
- Google Sheets에서 CSV 다운로드: 파일 → 다운로드 → 쉼표로 구분된 값(.csv)
- CSV 파일을 프로젝트 폴더에 저장

### 2. CSV 구조

**방법 A: 헤더 있는 형식**
```csv
Location,Team,ID,Age
웨슬리홀,새1,이민재6550,34
웨슬리홀,새1,김정민7823,28
칼빈채플,새4,김강민6324,40
자모영아실,여6,신은혜7567,43
```

**방법 B: 헤더 없는 형식 (4주차 이후)**
```csv
웨슬리홀,청1,송유미2205,34
웨슬리홀,청1,곽빛보라2672,38
칼빈채플,남4,전자룡7684,42
자모영아실,여6,신은혜7567,43
```

**필수 컬럼:**
- `Location` 또는 1번째 컬럼: 위치 (웨슬리홀, 칼빈채플, 자모영아실)
- `Team` 또는 2번째 컬럼: 조 이름 (새1, 남2, 여3, 청1 등)
- `ID` 또는 3번째 컬럼: "이름+전화번호4자리" 형식 (예: 이민재6550, 송유미2205)
- `Age` 또는 4번째 컬럼: 나이 (숫자, 비어있으면 0으로 처리)

### 3. 자동 업데이트

**방법 A: 헤더 있는 CSV 파일 (기존 방식)**
```bash
python update_data.py
```
또는 특정 파일 지정:
```bash
python update_data.py "교리교육 출석부_251115.csv"
```

**방법 B: 헤더 없는 CSV 파일 (4주차 이후)**
```bash
python convert_csv_4week.py
```
스크립트가 `출석부작업_251122 - admin_4주차.csv` 파일을 읽어서 `data.json`을 업데이트합니다.

**다른 파일명 사용 시:**
`convert_csv_4week.py` 파일에서 파일명을 수정하거나, 스크립트를 복사하여 파일명을 변경하세요.

실행하면 `data.json` 파일이 자동으로 업데이트되고, 통계가 출력됩니다.

### 4. 업데이트 검수

업데이트가 정상적으로 완료되었는지 확인:

```bash
python check_update.py
```

이 스크립트는 다음을 자동으로 확인합니다:
- ✅ CSV와 JSON의 총 인원 수 비교
- ✅ 위치별 인원 수 비교 (웨슬리홀, 칼빈채플, 자모영아실)
- ✅ 조별 인원 수 비교
- ✅ 나이 통계 비교 (60세 이상, 50-59세, 50세 미만, 나이 미입력)
- ✅ 데이터 매칭 확인 (CSV에만 있거나 JSON에만 있는 항목)

차이가 발견되면 상세한 내용이 출력됩니다.

📖 **자세한 설명**: `UPDATE_GUIDE.md` 파일을 참고하세요.
